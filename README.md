# Bingo Image Game Show

A fully interactive, web-based Bingo game designed for events, workshops, and social gatherings. 
<br> This tool allows users to create custom Bingo games using their own images and generate printable cards instantly.

## Key Features
* **Custom Image Upload**: Upload any set of images to use as your Bingo pool.
* **Dynamic Grid Selection**: Supports grids from 3x3 up to 8x8.
* **PDF Card Generator**: Built-in tool to generate and download multiple unique, printable Bingo cards.
* **Smart Image Calculator**: Helps you determine how many images you need based on the number of participants and grid size.
* **Bilingual Support**: Full support for both English and Hebrew (RTL/LTR) with a single click.
* **Interactive UI**: Features smooth animations, keyboard shortcuts, confetti and sound effects.
* **Caption Mode**: Toggle image filenames as captions on the screen during the draw.

## How to Use
1. **Set the Grid**: Choose your desired grid size (e.g., 5x5).
2. **Upload Images**: Click the upload button to select images from your computer. 
3. **Generate Cards**: Use the PDF icon to create and download printable cards for your players.
4. **Start the Game**: Click the "Draw" button or press the Spacebar to randomly select the next image.
